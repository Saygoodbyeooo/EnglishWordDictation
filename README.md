## 📖 项目简介
**EnglishWordDictation** 是一款专为英语学习打造的**纯前端、轻量化、高度自定义**智能单词默写工具。无需服务器、无需安装软件，打开即用。支持自定义词库导入、智能分页、高效默写交互、专业打印适配，完美适配个人学习、课堂教学、自主刷题等全场景需求。

致力于打造极简、高效、无干扰的英语单词默写体验，让单词记忆更高效、学习更专注。

## ✨ 核心特性
### 🎯 核心功能
- **自定义词库导入**：支持 TXT 文本批量导入单词，一键解析生成默写表
- **标准化词库模板**：`英文=中文` 极简格式，零学习成本
- **智能自适应分页**：自动适配任意数量单词，批量单词自动分页展示
- **高效交互体验**：Tab 键快捷切换输入框，自动跳过复选框，极致默写流畅度
- **本地数据持久化**：浏览器本地存储默写答案，刷新不丢失，随时继续学习
- **答案管控**：一键显示/隐藏标准答案，自主检测学习成果

### 🖨️ 专业打印优化
- 打印自动隐藏操作栏、复选框、提示文字
- 默写区域纯空白展示，无冗余文字
- 支持**全量单词打印**与**勾选单词定向打印**
- 适配A4纸张，打印排版整洁美观

### 🛠️ 便捷操作
- 一键全选/取消全选单词
- 一键清空所有内容与本地数据
- 响应式布局，适配电脑端全尺寸屏幕

## 🚀 快速开始
### 1. 极简使用（无需部署）
1. 下载项目源码
2. 直接用浏览器打开 `index.html` 文件
3. 立即使用，无需配置环境、无需依赖安装

### 2. 自定义词库导入
1. 新建文本文档（`.txt`）
2. 按照**标准格式**编写单词：
   ```txt
   pyramid=金字塔
   graph=图表；曲线图
   establish=建立；创立
   ```
3. 点击页面「选择文件」，导入TXT文件
4. 自动生成专属单词默写表

## 📝 使用说明
### 词库格式规范
- 每行一个单词，格式：`英文单词=中文释义`
- 自动忽略空行、多余空格，容错性强
- 支持任意数量单词（1~1000+）

### 核心操作
- **Tab 键**：在答案输入框间快速切换，自动跳过复选框
- **打印默写表**：一键生成空白默写纸，支持全量/勾选打印
- **保存答案**：本地存储默写内容，关闭浏览器不丢失
- **显示答案**：对照标准答案，快速核对纠错

## 🧰 技术栈
- **核心语言**：HTML5 + CSS3 + Vanilla JavaScript (原生JS)
- **存储方案**：浏览器 LocalStorage 本地持久化
- **文件解析**：原生 FileReader API 实现 TXT 导入
- **样式设计**：响应式布局 + 专业打印媒体查询
- **交互优化**：Tab 索引控制、表单快捷操作
